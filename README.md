# ✈️ British Airways Flight Operations Dashboard

An interactive **Microsoft Excel Dashboard** developed to analyze British Airways flight operations and provide meaningful business insights using Excel's data analysis and visualization features.

---

## 📌 Project Overview

This dashboard helps monitor flight operations by analyzing:

- Total Flights
- Countries Served
- Top Destination
- Total Seat Capacity
- Eligible Passengers
- Aircraft Utilization
- Passenger Tier Analysis
- Flights by Arrival Region
- Flights by Arrival Country
- Seat Capacity by Aircraft Type
- Flights by Time of Day

Interactive slicers allow users to filter data by:

- Flight Date
- Haul Type
- Arrival Region
- Aircraft Type
- Arrival Country
